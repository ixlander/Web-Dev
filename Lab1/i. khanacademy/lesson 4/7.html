First Steps into Programming

Posted on May 15, 2023

Today marks the beginning of my exciting journey into the world of programming! I've always been fascinated by how websites and applications work, and I've finally decided to take the plunge and learn how to create them myself.

I started by researching different programming languages and decided to begin with web development. HTML, CSS, and JavaScript seem like the perfect trio to start with. I'm looking forward to building my first web pages and seeing them come to life in the browser.

My goals for the next few weeks:

  • Learn the basics of HTML
  • Understand how CSS works
  • Create a simple personal website

I'm both nervous and excited about this new adventure. Here's to the start of something amazing!

Learning HTML and CSS

Posted on June 1, 2023

It's been a couple of weeks since I started my programming journey, and I'm happy to report that I've made significant progress with HTML and CSS!

I've learned how to structure a web page using HTML tags like <header>, <nav>, <main>, and <footer>. It's amazing how these simple tags can create a solid foundation for a website.

CSS has been a bit more challenging, but also incredibly rewarding. I've learned about:

  • Selectors and how to target specific elements
  • The box model and how it affects layout
  • Flexbox for creating responsive designs
  • Color theory and how to choose a pleasing color scheme

I'm particularly proud of this little CSS snippet I wrote to create a gradient background:


body {
    background: linear-gradient(to right, #3498db, #2c3e50);
    color: white;
}

Next up: JavaScript! I can't wait to add interactivity to my web pages.

Diving into JavaScript

Posted on June 15, 2023

Wow, time flies when you're having fun coding! I've now started my journey into JavaScript, and it's opening up a whole new world of possibilities.

JavaScript is quite different from HTML and CSS. It's a full-fledged programming language that allows me to add interactivity and dynamic content to my web pages. Here are some key concepts I've learned so far:

  • Variables and data types
  • Functions and how to use them
  • Conditional statements (if/else)
  • Loops for repeating actions
  • DOM manipulation to change HTML elements

I'm particularly excited about event listeners. They allow me to make my web pages respond to user actions. Here's a simple example I wrote:


document.getElementById('myButton').addEventListener('click', function() {
    alert('Hello, World!');
});

This code displays an alert when a button is clicked. It's a small step, but it feels like magic to see my web page come alive with interactivity!

My next goals are to learn about APIs and how to fetch data from servers. I'm also planning to start a small project to put all these new skills into practice. Stay tuned for more updates on my programming journey!
